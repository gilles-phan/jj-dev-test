# Test Technique - Développeur Frontend Angular

## Lancement du projet
Récupérer la dernière version, téléchargez les dépendances (`npm i`), lancez les tests (`ng test`), puis lancer le projet (`ng serve`).

Rappel : Le code est dispo sur ce [dépôt Github](https://github.com/gilles-phan/jj-dev-test).

## Build
Projet Angular Classique, donc `ng build`.

## Version du projet
J'ai installé la dernière version d'Angular pour le test, mais j'ai respecté les consignes demandées (comme l'utilisation des modules à la place des composants standalone).

J'ai ajouté un tag pour chaque partie, pour pouvoir se retrouver plus facilement. 

## Tests
Les tests unitaires ne sont pas exaustifs, mais suffisant dans le cadre d'un test technique.
